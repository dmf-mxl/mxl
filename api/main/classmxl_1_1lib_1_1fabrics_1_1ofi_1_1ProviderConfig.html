ProviderConfig mxl::lib::fabrics::ofi::ProviderConfig::create ( Provider  provider,
bool  isTarget,
std::optional< ProviderCapabilities >  capabilities 
)
static

Create a provider config for the given provider. When capabilities is nullopt (query path), caps and requiredCaps are zero so no fi_info entries are rejected on capability grounds. When present (setup path), libfabric caps are derived from the MXL capability flags.

Parameters
isTargettrue for target (FI_REMOTE_WRITE), false for initiator (FI_WRITE).
