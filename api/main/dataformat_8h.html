◆ MXL_DATA_FORMAT_GRAIN_SIZE

#define MXL_DATA_FORMAT_GRAIN_SIZE   4096

Fixed size in bytes of one MXL video/smpte291 data format grain. This is large enough to hold all the ANC data in a single grain. This size is a usual VFS page; no point in going smaller.
