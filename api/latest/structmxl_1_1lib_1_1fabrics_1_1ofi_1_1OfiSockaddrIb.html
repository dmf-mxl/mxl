Detailed Description

Local declaration of libfabric's internal struct ofi_sockaddr_ib.

The public libfabric headers do not expose this type, so we mirror its layout byte-for-byte from libfabric's include/ofi_net.h to interpret FI_SOCKADDR_IB addresses exactly like ofi_straddr.
